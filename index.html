GluTraductor

Este es un proyecto que comencé como una broma para hablar como un pez en Twitter, pero al final se convirtió en un proyecto realicé en Python para traducir palabras del español al pez. Añadí la librería TKinter para crear una interfaz gráfica. Quise implementar más idiomas como el inglés, italiano, etc., que finalmente realicé con la API de Google Translate. Finalmente, incluso añadí el aurebesh de Star Wars como lenguaje. Más adelante me gustaría implementar más idiomas y mejorar la interfaz gráfica. Cuando lo haga, lo publicaré en GitHub.
